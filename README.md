# Pickly

Pickly는 친구들의 반응과 AI 분석을 바탕으로, 가장 잘 나온 사진을 빠르게 찾아주는 사진 선택 플랫폼입니다.
프론트엔드 주소: https://pickly-eta.vercel.app/
## 프로젝트 소개

사진 촬영 후, 여러 장의 사진 중에서 어떤 컷이 가장 좋을지 고민한 경험이 있나요?
Pickly는 친구들의 반응과 AI 기반 분석을 통해, 팀원들이 자연스럽게 선택한 베스트컷을 추천해줍니다.


## 핵심 기능

- 친구들의 반응 확인
  - 여러 명이 사진에 대해 좋아요, 코멘트, 선호도를 남길 수 있습니다.
- AI 기반 분석
  - 얼굴 표정, 구도, 분위기, 인물 배치 등을 분석해 후보 사진을 평가합니다.
- 베스트컷 추천
  - 사람들의 반응과 AI 점수를 종합해 가장 적합한 사진을 제안합니다.
- 빠른 의사결정
  - 사진 선택 과정을 더 간단하고 즐겁게 만들어 줍니다.

## 왜 Pickly인가

사진은 단순히 많이 찍는 것이 아니라, 사람들의 반응과 분위기를 함께 반영할 때 더 좋은 선택이 됩니다.
Pickly는 “누가 뭐라고 했는지”와 “AI가 어떻게 판단했는지”를 함께 보여주며, 더 합리적이고 재미있는 선택 경험을 제공합니다.

## 기대 효과

- 사진 선택 시간을 단축할 수 있습니다.
- 친구들의 의견을 한눈에 확인할 수 있습니다.
- AI 분석으로 더 객관적인 선택을 도울 수 있습니다.

## 향후 방향

- 친구 추천 기능 고도화
- AI 분석 결과 시각화 개선
- 사진 앨범/공유 기능 확장
